  ![](https://sta-op.douyucdn.cn/butterfly-java/2022/08/29/350e4c1e28dbbf4214ca60722726c7a9.png)

● ESLint 规范

● 注释规范
规范详情可参见：[](https://www.yuque.com/fengyinchao/awb51o/ksm7sc)
